Название: Homeet_

Описание: Для хакатона, организованного НИУ ВШЭ и PIX ROBOTICS, наша команда разработала веб-сайт, который отображает расписание для факультета креативных индустрий в зависимости от выбранной группы.

Название: Python_OOP
Наиболее используемый язык: Python
Описание: Эта программа моделирует работу сотрудника и производит расчет его заработной платы, включая базовую зарплату и бонус. Она также генерирует расчетный листок, который содержит информацию о сотруднике, его базовой зарплате и зарплате с учетом бонуса.
